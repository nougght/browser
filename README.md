# Кроссплатформенный браузер (Windows, Linux, Android)
### Структура браузера:

- ядро (С++, SQLite);
- десктоп клиент (Qt/C++);
- андроид клиент (Kotlin) - *в планах после завершения ядра*;

#### В качестве движка используются QtWebEngine(Chromium) и Android WebView.
#### Ядро браузера работает в отдельном потоке со своим eventloop и взаимодействует с ui через события.
